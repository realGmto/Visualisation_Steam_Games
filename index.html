About

This is a small data visualization project which has a purpose of showing some basic ways of visualizing data while also adding some interactivity to it. Data that was used is Steam Games Dataset which came from kaggle. Data has been processed so it would remove some useless columns and also reduce its size by dropping games that are older than 2012.

How to use

Usage is pretty simple. When you get to the dashboard you will see that graphs are already created using all data. Interactivity options are:
- tooltip shows when aiming at the graph element which has additional information.
- clicking at the element will add it to the filter list which will be used to extract specific data and update graphs
- Clicking at the added filter will remove it from the filter list
- Clicking Apply will apply filters and update graphs
- Clicking remove will remove all filters and update graphs
